Ping-Display

Un mélange de projet informatique et electronique puisque ce projet consistait en la création d'un petit appareil basé sur l'ESP8266 avec un écran qui ping le serveur 1.1.1.1, mesure le temps de réponse et affiche le résultat sur un écran OLED ainsi qu'un histogramme. Code réalisé en C++ sur l'arduino IDE.

Weather-Station-Project

Encore un mélange informatique / electronique. Ici le but etait de creer un petit appareil qui lis les etats eletriques de capteurs et renvoie les infols lues par mqtt. Avec une implantation low power pour finir ce projet. Code réalisé en C++ sur l'arduino IDE.
